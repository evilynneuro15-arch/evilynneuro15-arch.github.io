Neuro-Bot — Privacy Policy

Last updated: 9 January 2026

This Privacy Policy explains how Neuro-Bot handles user data.

1. Data Collected

Neuro-Bot collects only the minimum data required for its features:

Data and its Purpose for the bot
Discord User ID - To associate preferences such as city, currency, or game watchlists
City name - For weather commands
Currency preference - For Steam price localization
Game watch entries - To notify users of sales

No message content is stored.

2. Data Storage

All data is stored locally on the bot host system in JSON files and is never sold or shared with third parties. Nor is it public facing

3. Data Retention

User data is retained only while the user continues to use the Bot.
Users may request deletion of their data at any time.

4. Third-Party Services

Neuro-Bot interacts with:

OpenWeather API — weather data

Steam Store API — game price & metadata

No personal Discord information is shared with these services.

5. Data Security

Reasonable measures are taken to protect stored data; however, no system is guaranteed to be completely secure.

6. User Rights

Users may request:

A copy of stored personal data

Complete deletion of stored data

Requests can be made by contacting the bot owner.

7. Changes

This Privacy Policy may be updated at any time. Continued use of the Bot constitutes acceptance of the latest version.
