# Neuro-Bot — Privacy Policy

**Last updated:** 9 January 2026

This Privacy Policy explains how Neuro-Bot handles user data.

---

## 1. Data Collected

Neuro-Bot collects only the minimum data required for its features:

| Data | Purpose |
|------|---------|
| Discord User ID | Associate user preferences (city, currency, game watchlists) |
| City name | Provide weather commands |
| Currency preference | Localize Steam prices |
| Game watch entries | Notify users of sales |

**No message content is stored.**

---

## 2. Data Storage

All data is stored locally on the bot host system in JSON files and is:

- Never sold or shared with third parties  
- Not publicly accessible  

---

## 3. Data Retention

- User data is kept only while the user continues to use the bot.  
- Users may request deletion of their data at any time.

---

## 4. Third-Party Services

Neuro-Bot interacts with:

1. **OpenWeather API** — weather data  
2. **Steam Store API** — game pricing and metadata  

No personal Discord information is shared with these services.

---

## 5. Data Security

Reasonable technical measures are taken to protect stored data; however, no system is guaranteed to be completely secure.

---

## 6. User Rights

Users may request:

- A copy of stored personal data  
- Complete deletion of stored data  

Requests can be made by contacting the bot owner.

---

## 7. Changes

This Privacy Policy may be updated at any time. Continued use of the bot constitutes acceptance of the latest version.
